Function nds::dma::fill

pub fn fill<T>(value: T, dst: &mut [T])where
    T: Sized + Copy,
Expand description

Fills dst by copying value using DMA channel 3. Copies src into dst using DMA channel 3. Calls wait_for on Channel::Ch3(Channel 3), so if interrupt DMA3 is disabled a panic may occur. Panics if size_of::<T>() is neither 2 nor 4.
